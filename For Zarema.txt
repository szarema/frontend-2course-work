Описание api для взаимодействия с сервером:

1. Создание таблицы в базе данных
    POST /api/table/create-table
    Тело запроса должно быть в формате JSON. Обязательные ключи:
        "tableName" - название таблицы
        "columnInfos" - список информации о колонках в таблице
        "primaryKey" - заголовок столбца, являющийся первичным ключём

    Возможные ответы на запрос:
        200 ОК - таблица успешно создана.
        406 NOT ACCEPTABLE - таблица с таким именем уже есть в БД/неверный формат тела запроса.
                             Среди заголовков есть "Error message", который хранит информацию о
                             конкретной ошибке.
        500 INTERNAL SERVER ERROR - произошло что-то, не предусмотренное мной.

2. Получение информации о таблице в базе данных
    GET /api/table/get-table-by-name/{name}
    Тело запроса отсутствует.

    Возможные ответы на запрос:
        200 ОК - таблица успешно создана или таблицы с таким названием нет в базе данных.
                 В случае, если таблица существует, тело ответа будет содержать строку в
                 формате JSON. Ключи:
                    "tableName" - название таблицы
                    "columnsAmount" - количество колонок в таблице
                    "primaryKey" - название колонки, хранящей первичный ключ
                    "columnInfos" - массив JSON строк, описывающих колонки в формате
                                    "НАЗВАНИЕ КОЛОНКИ":"ТИП ДАННЫХ В КОЛОНКЕ" (заглавными буквами)
                    Порядок ключей НЕ определён.
                 В случае, если таблица отсутсвует в базе данных тело запроса будет пустым.
        500 INTERNAL SERVER ERROR - произошло что-то, не предусмотренное мной.

3. Удаление таблицы
    DELETE /api/table/drop-table-by-name/{name}
    Тело запроса отсутствует.

    Возможные ответы на запрос:
        200 ОК - таблица успешно удалена из БД.
        406 NOT ACCEPTABLE - таблицы с таким именем в базе данных нет.
                             В заголовке ответа "Error message" описание ошибки.
        500 INTERNAL SERVER ERROR - произошло что-то, не предусмотренное мной.

4. Создание запроса, не привязанного к таблице
    POST /api/single-query/add-new-query
    Тело запроса должно быть в формате JSON. Обязательные ключи:
            "query" - SQL запрос к базе данных

    Возможные ответы на запрос:

                Тело ответа содержит число, являющееся id запроса в БД.
            400 BAD REQUEST - неверный формат тела запроса.
                              В заголовке ответа "Error message" описание ошибки.
            500 INTERNAL SERVER ERROR - произошло что-то, не предусмотренное мной.
